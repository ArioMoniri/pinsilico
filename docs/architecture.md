# Architecture

> Living document. Updated each phase. As of Phase 0 this is a stub; later phases populate it with sequence diagrams, IPC contracts, and the dual-view rendering pipeline.

## Overview

PInSilico is a three-layer system:

1. **Tauri 2.x shell (Rust)** — owns the OS window, file dialogs, and the sidecar process lifecycle. Talks to the webview over Tauri IPC and to the sidecar over `http://127.0.0.1:<ephemeral-port>` with a per-launch `X-Pinsilico-Token` header.
2. **Webview (React 18 + TypeScript)** — renders the UI and both 3D views (abstract arena via R3F/Three.js, atomistic via Mol*). State lives in Zustand stores; server calls go through TanStack Query against a typed client generated from the sidecar's OpenAPI 3.1 schema.
3. **Python 3.11 sidecar (PyInstaller)** — wraps the chemistry stack (RDKit, Biopython, biotite, Open Babel), the docking engines (Smina, Vina, optional DiffDock and Boltz-2), fpocket, and the simulation engine.

## Process model

- Shell spawns sidecar at startup. Sidecar prints `PINSILICO_TOKEN=…` and `PINSILICO_PORT=…` on stdout (Phase 1 introduces the token; Phase 6 wires the spawn into the Tauri shell).
- Shell health-checks `/health` until ready, then mounts the webview.
- Webview reads the API base + token from Tauri IPC commands (`get_api_base()`, `get_token()`).
- On window close: shell sends `POST /shutdown` to the sidecar, then kills the process after a 1 s grace period.

## What's shipped through Phase 10

**Sidecar** (Python 3.11, ~3 700 lines, 244 tests, 92 % coverage)

- Auth: per-launch token printed on stdout, `secrets.compare_digest` checks
- Logging: structlog JSON to stdout + rotating file (10 MB × 5 backups)
- Errors: standard `{error: {code, message, details}}` envelope on every non-2xx
- IO parsers: Biopython PDB + RDKit SDF/SMILES with round-trip property tests
- DB clients: RCSB, UniProt, AlphaFold, ChEMBL, PubChem, DrugBank
  (all HTTP respx-mocked; nightly live-network job in Phase 12)
- Pocket detection: fpocket subprocess wrapper with `_info.txt` +
  `pocketN_vert.pqr` parsing
- Docking: Smina + AutoDock Vina (shared adapter), DiffDock + Boltz-2
  weights-gated stubs that surface `ENGINE_NOT_AVAILABLE` until downloaded
- Simulation engine: overdamped Langevin Brownian step, Boltzmann residence
  via `τ = τ₀ · exp(-ΔG/RT)`, competition mode, fast-forward categorical
  sampling, deterministic under seed
- HTTP API: 22 routes under `/db/*`, `/pocket/*`, `/sim/*`, plus `/health`,
  `/version`, `/shutdown`. OpenAPI 3.1 schema auto-generated by FastAPI;
  Phase 12 packaging adds the `openapi-typescript` regen step
- Session bundle: `.pinsilico` zip with deterministic bytes
  (identical input → identical archive)

**Tauri shell** (Rust)

- `app_version()` + `window_title()` helpers anchoring the six-way version
  sync (Cargo.toml, package.json, tauri.conf.json, pyproject.toml,
  version.ts, `__init__.py`)
- `sidecar` module: stdout-banner parser + `SidecarHandle` with
  loopback-only host validation. 10 unit tests on the parser cover
  uvicorn-line interleaving, missing keys, bad ports, non-loopback rejection

**Frontend** (React 18, TypeScript 5, 44 tests)

- Typed `PinsilicoClient` matching every Phase 5 route, `ApiError` with
  `(status, code, details)` extracted from the standard envelope
- Zustand stores: `useSessionStore` (proteins / ligands by id),
  `useSceneStore` (view mode + active protein / pocket + paused flag)
- R3F abstract arena: `ProteinShell`, `PocketMarker` (real fpocket
  centroids, druggability-tinted halo), `ParticleSwarm` (single
  InstancedMesh for the swarm)
- Mol* atomistic seam: lazy-loaded `MolstarViewer` + pure-function
  `representationFor(view)` preset + `interpolateLigandFrames` with the
  bound-state-snaps-to-position invariant
- Panels: `ProteinPanel` (list / add / remove / role), `SimPanel` (every
  BUILD_PROMPT §1 P7 knob: mode / concentrations / iterations / seed /
  temperature / animation style / encounter-potential toggle /
  playback speed)

**CI**

- `.github/workflows/ci.yml` runs 10 jobs on every push (Lint +
  Python/Frontend/Rust × Ubuntu/macOS-14/Windows). All green at the
  most recent tagged commits.
